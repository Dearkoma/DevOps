## 解答"为何不启动 React 也能进前端"
- 用户疑问：没启动 React，为何访问 Spring Boot 的 8080 能看到前端页面。
- 解释：本项目用 build 模式，`npm run build` 把 React 源码编译成静态文件扔进 `static/`，Spring Boot 内置 Tomcat 自动托管。所以 8080 既是 API 服务器也是前端静态文件服务器，不需要 Vite 开发服务器参与。
- 用 SVG 对比图说明了两种模式：开发模式（npm run dev → 5173/3000，HMR）vs 本项目模式（npm run build → static → 8080 托管）。

## 切换到开发模式
