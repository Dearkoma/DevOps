### 认证流程
1. 用户访问任意页面 → ProtectedRoute 检查 token → 无 token 跳转 /login
2. Login 页检测后端状态（fetch /api/auth/me）→ 离线显示启动提示
3. 提交登录 → POST /api/auth/login → 获得 JWT token → 存入 localStorage → 跳转 /dashboard
4. 后续所有 API 请求自动带 Authorization: Bearer <token>
5. 401 响应 → 自动清除 token → 跳转登录页
6. 侧边栏底部显示当前用户 + 退出按钮 → 退出清空 token → 跳转登录
